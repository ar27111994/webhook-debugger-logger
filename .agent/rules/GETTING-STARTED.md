# Before You Begin

1. Install Apify CLI and run `apify login`
   [Apify CLI Installation](https://docs.apify.com/cli/docs/installation)
2. Install Chrome DevTools MCP
   [Chrome DevTools MCP](https://github.com/ChromeDevTools/chrome-devtools-mcp?tab=readme-ov-file#getting-started)
3. Run Codex or Claude Code, if available and start vibe coding your actor
4. Use the `apify` MCP Server (if available) to access and research the Apify documentation.
